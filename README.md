📚 Quick Reference Desk
Digital Masterpiece by Zaw Aung Htut (Library Science)
ဤပရောဂျက်သည် မြန်မာ့စာကြည့်တိုက်လောက၏ ဂန္ထဝင်မြောက်လှသော "မြန်မာရည်ညွှန်း" ပညာရပ်ကို အခြေခံ၍ ခေတ်သစ်နည်းပညာဖြင့် ပြန်လည်အသက်သွင်းထားသော Interactive Virtual Reference Desk တစ်ခု ဖြစ်ပါသည်။ စာကြည့်တိုက်သို လူကိုယ်တိုက် လာရောက်ရန်မလိုဘဲ အွန်လိုင်းမှတစ်ဆင့် သတင်းအချက်အလက်များကို ချက်ချင်း (Quick Reference) ရှာဖွေပေးနိုင်ရန် ရည်ရွယ်ပါသည်။
🌟 အဓိက လုပ်ဆောင်ချက်များ (Key Features)
1. ဘာမေးမေး အဖြေပေးစနစ် (Interactive Search)
 * Quick Reference: အဘိဓာန်၊ စွယ်စုံကျမ်း၊ လမ်းညွှန် နှင့် စာစုစာရင်း စသည့် အခြေခံရည်ညွှန်းပညာရပ်ဆိုင်ရာ မေးခွန်းများကို ချက်ချင်းအဖြေပေးနိုင်ခြင်း။
 * Long-range Reference: ရှာဖွေရခက်ခဲသော မေးခွန်းများအတွက် စာကြည့်တိုက်မှူးနှင့် တိုက်ရိုက်ချိတ်ဆက်နိုင်ရန် အချက်အလက်တောင်းယူခြင်း။
2. အဆင့်မြင့် ဖတ်ရှုမှုစနစ် (Smart Reading)
 * Dark/Light Mode: အမှောင်နှင့် အလင်း မုဒ်များကို စိတ်ကြိုက်ပြောင်းလဲနိုင်ခြင်း (Shortcut: Ctrl+D)။
 * Adjustable Font: စာလုံးအရွယ်အစားကို စိတ်ကြိုက်ညှိနိုင်ခြင်း (Shortcut: Ctrl++/-)။
 * Progress Bar: ဖတ်ရှုမှုအခြေအနေကို Real-time ပြသပေးခြင်း။
3. လက်လှမ်းမီမှုနှင့် မျှဝေမှု (Accessibility & Sharing)
 * Text-to-Speech: စာသားများကို အသံထွက်ဖတ်ပေးနိုင်သောကြောင့် အမြင်အာရုံချိုယွင်းသူများအတွက်ပါ အဆင်ပြေစေခြင်း။
 * Professional Printing: ရည်ညွှန်းဒေတာများကို ပုံစံတကျ ပုံနှိပ်ထုတ်ယူနိုင်ခြင်း။
🛠️ အသုံးပြုထားသော နည်းပညာများ
 * Frontend: HTML5, CSS3, JavaScript (Single Code Architecture)
 * Framework: Custom Native JS (Lightweight & Fast)
 * Accessibility: Web Speech API
👤 စာရေးသူအကြောင်း (About Author)
ဦးဇော်အောင်ထွတ် (ခ) ဇော်အောင်ထွတ် (စာကြည့်တိုက်ပညာ)
 * သိပ္ပံဘွဲ (ဓာတုဗေဒ) နှင့် မဟာဝိဇ္ဇာ (စာကြည့်တိုက်နှင့် သုတပညာ) ဘွဲရရှိထားသူ။
 * ပခုက္ကူဦးအုန်းဖေ စာပေဆု (၁၉၉၇) နှင့် အကောင်းဆုံး စာတမ်းဆု (၂၀၁၅) ရရှိထားသူ။
 * စာကြည့်တိုက်ပညာနှင့် နည်းပညာပေါင်းစပ်မှုများကို နှစ်ပေါင်းများစွာ လေ့လာပြုစုနေသူ။
